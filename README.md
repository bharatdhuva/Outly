# Outly 🚀 | AI Job Hunt Automation Tool

**Outly automates your entire job search** — Deep company research karke hyper-personalized cold emails likhta hai, resume ko target job ke hisaab se tailor karta hai, aur LinkedIn + Twitter (X) pe smart networking posts auto-generate + schedule kar deta hai.

---

### ✨ Features

- **AI Cold Email Generator** — CSV se company names leke GPT-4 research karta hai aur unique personalized emails banata hai
- **Gmail Auto-Draft** — Saare emails Gmail mein automatically drafts mein save ho jaate hain (Gmail API + OAuth 2.0)
- **AI Resume-to-JD Matcher** — GPT-4 resume aur Job Description ko score karta hai, missing ATS keywords batata hai aur suggestions deta hai
- **Fault-Tolerant Job Queue** — Bull + Redis se rate-limiting, auto-retry aur 100% reliable execution
- **AI Content Scheduler** — LinkedIn + Twitter posts GPT-4 se generate karke Telegram Bot ke through schedule kar sakta hai
- **Production Ready** — Clean architecture, error handling aur scalable design

---

### 🛠️ Tech Stack

**Frontend:** React.js  
**Backend:** Node.js + Express.js  
**AI:** GPT-4 (OpenAI)  
**Databases & Queues:** Redis + Bull Queue  
**APIs:** Gmail API, OAuth 2.0, Telegram Bot API  
**Others:** TypeScript, JWT, MongoDB (optional)
